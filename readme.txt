Quickly and easily add Latakoo Flight videos to your blog posts.

== Description ==

This plugin lets you simply add videos stored on your latakoo flight account
to your wordpress blog.

This was mainly written to scratch an itch.

Once installed you will be able to link to latakoo videos by using the following
short code:

[latakoo]videourl or video ID[/latakoo]

You can also specify width and height of the player, e.g.

[latakoo width="xxx" height="yyy"]12345[/latakoo]

== Installation ==

1. Upload /wp-content/plugins/ directory
2. Unzip
3. Activate through the plugins menu
